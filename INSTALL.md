# 📲 Установка Memory Lane как приложение

## iOS (iPhone/iPad)

1. Откройте `index.html` в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите "На экран «Домой»"
4. Нажмите "Добавить"
5. Готово! Теперь Memory Lane на вашем домашнем экране

## Android (Chrome)

1. Откройте `index.html` в Chrome
2. Нажмите меню (три точки)
3. Выберите "Добавить на главный экран" или "Установить приложение"
4. Нажмите "Добавить"
5. Готово! Приложение установлено

## Desktop (Chrome/Edge)

1. Откройте `index.html` в Chrome или Edge
2. В адресной строке появится значок установки (⊕)
3. Нажмите на него
4. Нажмите "Установить"
5. Приложение откроется в отдельном окне

## Преимущества установки

- Работает как нативное приложение
- Быстрый доступ с домашнего экрана
- Полноэкранный режим (без браузерной панели)
- Все данные сохраняются локально

## Примечание

Для полноценной работы PWA необходимо:
- HTTPS соединение (или localhost для разработки)
- Service Worker (будет добавлен в следующей версии)
- Иконки приложения в папке `assets/`
