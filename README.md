# Memory Lane

Memory Lane — мобильное PWA-приложение для сохранения истории закрытых групп: друзей, семьи и команд.

Продукт помогает собирать общую историю группы в одном приватном пространстве: воспоминания, участников, достижения и места на карте.

## Что уже есть

- главный экран со списком групп и поиском;
- экран ленты событий внутри группы;
- экран участников;
- экран карты памяти;
- экран ачивок;
- создание групп и новых воспоминаний через модальные формы;
- экспорт и импорт данных через JSON;
- светлая и тёмная тема;
- локальное хранение данных в браузере.

## Что в планах

- AI-редактор описаний воспоминаний;
- полноценный offline PWA-режим с `service worker`.

## Текущий стек

- `Vite`
- `Vanilla JavaScript`
- `HTML5`
- `CSS3`
- `localStorage`
- `Leaflet` для карты

Проект находится в активной разработке. Основной SPA-сценарий уже реализован: можно управлять группами, воспоминаниями, участниками, картой и переносом данных. Вне текущей реализации остаются AI-редактор и полноценная offline PWA-поддержка.
