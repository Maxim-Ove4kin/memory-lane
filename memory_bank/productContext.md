# Product Context

## Проблема пользователя
Цифровая амнезия: тысячи фото, сотни шуток и идей исчезают в потоке чатов. Важные фото тонут в мемах, "то самое" видео невозможно найти, даты забываются.

## Решение
Memory Lane — приватное цифровое пространство для сохранения истории закрытых групп. Не соцсеть для лайков, а цифровой костёр вокруг которого собираются свои.

## Целевая аудитория
- **Друзья**: сохранение истории компании от студенческих вечеринок до свадеб
- **Семья**: цифровая летопись роста детей, семейных праздников
- **Команды**: фиксация пути компании, первые победы, факапы, корпоративы

## Ключевые ценности
- Приватность: доступ только по приглашениям
- Никаких алгоритмов, рекламы, посторонних глаз
- Офлайн-режим для походов
- Mobile-first: работает с телефона без установки
