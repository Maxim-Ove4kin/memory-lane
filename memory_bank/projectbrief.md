# Project Brief

## Обзор проекта
Memory Lane — это приватное PWA-приложение для сохранения истории закрытых групп (друзья, семья, команда). Представляет собой цифровой костёр, где сохраняются воспоминания: фото, видео, истории, места на карте, достижения участников.

## Цели проекта
1. Создать mobile-first пространство для хранения истории закрытых групп
2. Дать группе общий таймлайн воспоминаний с поиском и фильтрацией
3. Отображать участников, достижения и значимые места группы в едином интерфейсе
4. Поддержать перенос данных через импорт и экспорт JSON
5. Довести продукт до AI-assisted описаний и полноценного offline PWA-режима

## Текущий статус
Проект в активной разработке. Клиентское SPA на `Vite` уже покрывает основной пользовательский сценарий работы с группами, событиями, участниками, картой и переносом данных, но AI-интеграция и полноценный offline PWA-режим ещё не завершены.

Текущий product scope синхронизирован с `PRD.md`, а high-level архитектура фиксируется в `docs/README.md`.
