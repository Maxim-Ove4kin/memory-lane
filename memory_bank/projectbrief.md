# Project Brief

## Обзор проекта
Memory Lane — это приватное PWA-приложение для сохранения истории закрытых групп (друзья, семья, команда). Представляет собой цифровой костёр, где сохраняются воспоминания: фото, видео, истории, места на карте, достижения участников.

## Цели проекта
1. Создать мобильное-first PWA для хранения воспоминаний группы
2. Реализовать интерактивный таймлайн с хронологической структурой
3. Добавить карту памяти с пинами мест
4. Интегрировать AI-редактор для создания описаний
5. Реализовать профиль участника с тегами и ачивками от друзей

## Текущий статус
Проект в активной разработке. Клиентское SPA на `Vite` уже покрывает основной пользовательский сценарий работы с группами, событиями, участниками, картой и переносом данных, но AI-интеграция и полноценный offline PWA-режим ещё не завершены.
