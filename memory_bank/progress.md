# Progress

## Статус проекта
В активной разработке. Основной SPA-сценарий реализован, а `README.md`, `PRD.md`, `docs/README.md` и Memory Bank синхронизируются между собой; недостающими остаются AI-функции и полноценный offline PWA-режим.

## Готовые компоненты
- SPA shell на `Vite` с локальным хранением данных
- Экран списка групп с поиском и переходом внутрь группы
- Таймлайн событий с фильтрами по тексту, категории и году
- Экраны участников, ачивок и интерактивной карты памяти на `Leaflet`
- Создание групп и событий, импорт и экспорт JSON-данных
- Переключение темы и мобильная нижняя навигация

## В работе
- AI-редактор описаний
- Полноценная offline PWA-поддержка через `service worker`

## Known Issues
- Нет реального бэкенда — данные сохраняются только в `localStorage`
- AI-режим не реализован
- `manifest.json` присутствует, но `service worker` не реализован
- UI-документация в `memory_bank/ui_extension/pages/` была частично legacy и синхронизирована только в рамках текущей сессии
